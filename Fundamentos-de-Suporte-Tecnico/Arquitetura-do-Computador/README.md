# 💻 Arquitetura do Computador

Guia de estudos e documentação técnica com os principais conceitos apresentados no curso **Fundamentos do Suporte Técnico**, oferecido pelo **Google** na plataforma **Coursera**.

---

## 🎯 Objetivos

Neste módulo foram apresentados os conceitos fundamentais da arquitetura dos computadores, incluindo a representação de dados, componentes básicos, evolução da computação e conceitos essenciais utilizados na área de Tecnologia da Informação.

---

## 📖 Glossário

### 🧠 Abstração

Consiste em simplificar um sistema complexo para facilitar seu entendimento ou utilização.

### 📋 Algoritmo

Sequência ordenada de etapas utilizada para resolver um problema específico.

### 🔤 ASCII

Primeiro padrão amplamente utilizado para representar letras, números e símbolos através de códigos binários.

### 0️⃣1️⃣ Sistema Binário

Sistema numérico de base 2 utilizado pelos computadores para representar informações.

### 💾 Byte

Conjunto de 8 bits.

### 🔠 Codificação de Caracteres

Método utilizado para converter caracteres em valores binários.

### 🖥️ Computador

Dispositivo eletrônico capaz de armazenar, processar e transmitir informações.

### 🔒 Criptografia

Técnica utilizada para proteger informações contra acesso não autorizado.

### 🔢 Sistema Decimal

Sistema numérico de base 10 utilizado pelas pessoas.

### 🌐 Exclusão Digital

Diferença de acesso e conhecimento entre pessoas que possuem e que não possuem habilidades digitais.

### 💼 Tecnologia da Informação (TI)

Área responsável pelo uso de computadores, redes e sistemas para armazenar, processar e compartilhar informações.

### 🐧 Linux

Sistema operacional de código aberto amplamente utilizado em servidores, empresas e ambientes de desenvolvimento.

### ⚙️ Portas Lógicas

Circuitos eletrônicos que realizam operações lógicas utilizando transistores.

### 🌍 Código Aberto (Open Source)

Software cujo código-fonte pode ser estudado, modificado e distribuído livremente.

### 📱 PDA (Personal Digital Assistant)

Dispositivo portátil utilizado como assistente pessoal digital, considerado precursor dos smartphones.

### 🗂️ Cartões Perfurados

Meio antigo de entrada de dados em computadores utilizando cartões com perfurações.

### 🎨 Modelo RGB

Modelo de cores baseado na combinação de Vermelho (Red), Verde (Green) e Azul (Blue).

### 🌎 UTF-8

Padrão moderno de codificação de caracteres compatível com praticamente todos os idiomas.
