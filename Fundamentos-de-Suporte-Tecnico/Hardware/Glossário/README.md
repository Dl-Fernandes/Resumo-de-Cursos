# 📖 Glossário

## 🚌 Barramento de Endereço (Address Bus)

Conjunto de linhas de comunicação responsável por transportar os endereços de memória enviados pela CPU ao Controlador de Memória (MCC). Ele informa onde os dados estão localizados, mas não transporta os dados em si.

## 💾 ATA (Advanced Technology Attachment)

Padrão de interface utilizado para conectar dispositivos de armazenamento ao computador. Foi amplamente utilizado antes do SATA.

## 🖥️ ATX (Advanced Technology eXtended)

Formato de placa-mãe mais utilizado em computadores desktop, oferecendo boa capacidade de expansão.

## 🔄 Compatibilidade com Versões Anteriores (Backward Compatibility)

Capacidade de um hardware ou software moderno funcionar corretamente com tecnologias mais antigas.

## ⚡ Cache

Memória de alta velocidade utilizada para armazenar temporariamente dados acessados com frequência pela CPU, reduzindo o tempo de acesso às informações.

## 🔋 Ciclo de Carga

Processo completo de carga e descarga de uma bateria recarregável.

## ⏱️ Frequência de Clock

Sinal elétrico periódico utilizado para sincronizar as operações da CPU e dos demais componentes do computador.

## 🚀 Velocidade de Clock (Clock Speed)

Quantidade de ciclos de clock que a CPU consegue executar por segundo, normalmente medida em Gigahertz (GHz).

## 🔌 Clock Wire

Linha de comunicação responsável por transportar o sinal de clock que sincroniza os componentes do computador.

## 🔲 Soquete da CPU (CPU Socket)

Encaixe físico localizado na placa-mãe onde o processador é instalado.

## 📏 Tamanho de Dados

Unidades utilizadas para medir a quantidade de informações armazenadas ou transmitidas, como bit, byte, kilobyte, megabyte e gigabyte.

## 💽 DIMM (Dual Inline Memory Module)

Módulo de memória utilizado para instalar memória RAM em computadores desktop.

## 🖥️ DisplayPort

Interface digital utilizada para transmitir áudio e vídeo em alta qualidade entre computadores e monitores.

## ⚠️ Descarga Eletrostática (ESD)

Descarga repentina de eletricidade estática capaz de danificar componentes eletrônicos sensíveis.

## 🔄 Barramento Externo de Dados (External Data Bus - EDB)

Conjunto de linhas responsável pela transferência de dados entre a CPU, a memória RAM e outros componentes.

## 📐 Form Factor (Fator de Forma)

Padrão que define o tamanho físico, o formato e a disposição dos componentes de uma placa-mãe.

## 🌡️ Dissipador de Calor (Heat Sink)

Componente responsável por absorver e dissipar o calor gerado pelo processador.

## 📜 Conjunto de Instruções (Instruction Set)

Conjunto de comandos que um processador é capaz de interpretar e executar.

## 🖥️ ITX (Information Technology eXtended)

Família de formatos de placas-mãe compactas, utilizada em computadores de pequeno porte.

## 🍎 Lightning

Conector proprietário desenvolvido pela Apple para transmissão de energia e dados.

## 📡 Mb/s (Megabit por Segundo)

Unidade utilizada para medir velocidade de transferência de dados.

## 📺 Micro HDMI

Versão reduzida do conector HDMI utilizada em dispositivos portáteis.

## 📺 Mini HDMI

Versão compacta do conector HDMI utilizada em câmeras e outros dispositivos móveis.

## 🔌 Micro USB

Versão reduzida do conector USB utilizada em smartphones e dispositivos portáteis mais antigos.

## 🔌 Mini USB

Versão menor do conector USB utilizada em equipamentos eletrônicos antigos.

## 🚀 Overclocking

Técnica que consiste em aumentar a frequência de operação da CPU ou de outros componentes para obter maior desempenho.

## ✅ POST (Power-On Self-Test)

Teste automático executado durante a inicialização do computador para verificar se os principais componentes de hardware estão funcionando corretamente.

## 📝 Registros (Registers)

Pequenas áreas de armazenamento extremamente rápidas localizadas dentro da CPU e utilizadas durante o processamento de instruções.

## 💾 Chip ROM (Read Only Memory)

Memória não volátil onde normalmente fica armazenado o firmware do computador, como o BIOS ou o UEFI.

## 💿 RPM (Revoluções por Minuto)

Unidade utilizada para medir a velocidade de rotação dos discos rígidos (HDDs).

## 🌡️ Temperatura de Funcionamento Seguro

Faixa de temperatura recomendada para que um componente eletrônico opere sem risco de danos.

## 📦 SOC (System on a Chip)

Circuito integrado que reúne diversos componentes, como CPU, GPU, memória e controladores, em um único chip.

## 🔩 Standoffs

Pequenos espaçadores metálicos utilizados para fixar a placa-mãe ao gabinete, evitando contato direto com a estrutura metálica.

## 🔌 USB-C

Conector USB reversível capaz de transmitir dados, vídeo, áudio e energia através de uma única interface.
